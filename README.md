# Typing Dojo

Typing Dojo is a mobile-friendly Next.js app for practicing everyday Japanese phrases with Thai and English hints.

## Run locally

Prerequisite: Node.js

1. Install dependencies:
   `npm install`
2. Start the development server:
   `npm run dev`
3. Open `http://localhost:3000`

## Build

1. Run lint:
   `npm run lint`
2. Create a production build:
   `npm run build`

## Deploy to Vercel

1. Log in to Vercel if needed:
   `npx vercel login`
2. Link or create the project:
   `npx vercel link`
3. Deploy to production:
   `npx vercel --prod`
